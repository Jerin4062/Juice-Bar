# Simple Calculator

This is a simple calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic: +, -, ×, ÷
- Parentheses support
- Clear (C) and backspace (⌫)
- Responsive design
- Keyboard support

## Usage
1. Open `index.html` in your browser.
2. Click buttons or use your keyboard to input numbers and operators.
3. Press `=` or Enter to calculate.
